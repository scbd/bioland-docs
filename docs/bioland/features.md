# Bioland 2 Features

Discover the full capabilities of your Bioland 2 Clearing House Mechanism platform.

## Content Management

### Local Content

Create and manage a wide variety of biodiversity-related content:

| Content Type | Description |
|-------------|-------------|
| **News** | Announcements, updates, and press releases |
| **Events** | Meetings, workshops, conferences, and webinars |
| **Learning Resources** | Educational materials, guides, and training content |
| **Protected Areas** | Information about conservation sites and reserves |
| **Species** | Biodiversity data and species information |
| **Documents** | Publications, reports, and policy documents |
| **National Targets** | Progress towards CBD and biodiversity targets |

### Content Features

- **Rich Text Editor**: Format content with headings, lists, links, and media
- **Image Management**: Upload and manage images for your content
- **Categorization**: Organize content with tags and categories
- **Publication Workflow**: Draft, review, and publish content
- **Revision History**: Track changes and restore previous versions

## Global Interoperability

### Partner Content Integration

Bioland 2 connects with global biodiversity networks:

#### GBIF (Global Biodiversity Information Facility)
- Access to species occurrence data
- Integration with global biodiversity datasets
- Automatic updates from GBIF network

#### Panorama
- Solutions for a Healthy Planet
- Case studies and best practices
- Conservation success stories

#### CBD Secretariat
- News and events from the Secretariat
- National reports and submissions
- Global biodiversity framework updates

### CHM Interoperability

Connect with the global CHM network:

- **News Sharing**: Access news from other national CHMs
- **Event Coordination**: Discover events across the CBD community
- **National Targets**: Share progress on biodiversity targets
- **Resource Sharing**: Access shared learning resources

## Search and Discovery

### Advanced Search

Powerful search capabilities to find information quickly:

- **Full-text Search**: Search across all content
- **Content Type Filtering**: Narrow results by type (news, events, etc.)
- **Date Filtering**: Find content from specific time periods
- **Keyword Highlighting**: See where terms appear in results

### Dual Search Scope

Search across two distinct content pools:

| Tab | Content Source | Features |
|-----|----------------|----------|
| **Site Content** | Local content | Full filtering, all content types |
| **Secretariat** | Global CHM content | News, events, national targets |

### Search Tips

- Use **keywords** for broad searches
- Use **"quoted phrases"** for exact matches
- Results update **automatically** as you type

## Forum and Community

### Discussion Forums

Engage your community with integrated forums:

- **Create Topics**: Start discussions on biodiversity issues
- **Comments**: Enable community participation
- **Moderation**: Control forum content and users
- **Categories**: Organize discussions by theme

### Community Features

- User profiles and participation history
- Notification system for replies
- Content flagging and moderation tools
- Integration with main site content

## Multi-Language Support

### Translation Features

Reach a broader audience with multi-language support:

- **Multiple Languages**: Support for any number of languages
- **Linked Translations**: Translations connected to original content
- **Language Switcher**: Easy switching between languages
- **Partial Translation**: Publish content even if not fully translated

### Translation Workflow

1. Create content in the default language
2. Add translations for additional languages
3. Publish translations independently
4. Users select their preferred language

## Navigation and Menus

### Menu Management

Control your site's navigation structure:

- **Multiple Menus**: Main navigation, footer, sidebar
- **Nested Items**: Create dropdown/hierarchical menus
- **Custom Links**: Link to external resources
- **Content Links**: Link directly to site content
- **Ordering**: Control item position and hierarchy

### Menu Features

- Drag-and-drop reordering
- Visibility controls
- Multi-language menu items
- Responsive navigation for mobile devices

## User Management

### User Roles

Flexible role-based access control:

| Role | Capabilities |
|------|-------------|
| **Administrator** | Full system access, user management, configuration |
| **Editor** | Create, edit, and publish all content |
| **Contributor** | Create content, may require approval to publish |
| **Authenticated User** | Access restricted content, forum participation |
| **Anonymous** | View public content only |

### User Features

- User registration and approval workflows
- Password management and recovery
- Activity tracking and logging
- Permission customization
