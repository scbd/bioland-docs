## What is Bioland 2?

Bioland 2 is the national-level web application for the Convention on Biological Diversity (CBD) Clearing-House Mechanism (CHM). It provides countries with a modern platform to:

- **Access global CBD content** - View and search information from the central CBD knowledge base
- **Publish local content** - Create and manage country-specific biodiversity information
- **Organize information** - Build custom menu structures and navigation
- **Collaborate globally** - Share knowledge with the international biodiversity community
