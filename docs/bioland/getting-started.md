## What is Bioland 2?

Bioland 2 is a modern Clearing House Mechanism (CHM) platform designed for countries to share and manage biodiversity-related information under the Convention on Biological Diversity (CBD).

### Key Capabilities

- **Share biodiversity information** with national and international stakeholders
- **Connect with global networks** including GBIF, Panorama, and the CBD Secretariat
- **Manage local content** such as news, events, learning resources, and protected areas
- **Engage communities** through integrated forums and discussions
- **Support multiple languages** for broader accessibility

## Accessing Your Site

### Public Site Access

Your Bioland 2 site is publicly accessible at your organization's URL. Visitors can:

- Browse all published content
- Search local and global resources
- Access the forum and discussions
- View content in available languages

### Administrator Access

To manage content and settings, you'll need administrator credentials:

1. Navigate to your site's admin URL (typically `/admin`)
2. Enter your username and password
3. Click **Sign In**

::: tip
If you don't have admin credentials, contact your site administrator or the Bioland 2 support team.
:::

### First-Time Login

When logging in for the first time:

1. Use the temporary password provided to you
2. You'll be prompted to create a new password
3. Choose a strong, unique password
4. Save your credentials securely

## Navigating the Interface

### Public Site Navigation

The public site includes:

- **Main Menu**: Access to all content sections (News, Events, Resources, etc.)
- **Search**: Global search in the top-right corner
- **Language Switcher**: Change display language (if multiple languages are enabled)
- **Footer**: Links to additional resources and contact information

### Admin Interface

The admin interface provides access to:

- **Dashboard**: Overview of site activity
- **Publishing > Content**: Manage all content items
- **Structure > Menus**: Control site navigation
- **People**: User management
- **Configuration**: Site settings

## Your First Tasks

### 1. Explore the Public Site

Before making changes, familiarize yourself with your site:

- Visit the home page and navigate through all sections
- Try the search functionality
- View different content types (news, events, resources)
- Check the forum if enabled

### 2. Access the Admin Panel

Log in to the admin interface and explore:

- Review the dashboard for recent activity
- Navigate to Publishing > Content to see existing content
- Check the menu structure under Structure > Menus

### 3. Create Your First Content

Start with something simple:

1. Go to **Publishing > Content**
2. Click **Add Content**
3. Select a content type (e.g., News)
4. Fill in the title and body
5. Save as draft or publish immediately

### 4. Review Site Settings

Verify important settings:

- Site name and description
- Default language
- Contact information
- Social media links

## Troubleshooting

### Can't Access Admin Panel

- Verify you have the correct URL
- Check your username and password
- Ensure your account has admin privileges
- Contact your site administrator

### Forgot Password

1. Click the "Forgot password" link on the login page
2. Enter your email address
3. Check your email for reset instructions
4. Follow the link to create a new password

### Site Not Loading

- Check your internet connection
- Try a different browser
- Clear browser cache and cookies
- Contact your site administrator if the issue persists
