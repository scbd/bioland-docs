# Login

This guide explains how to log into the Biosafety Land system.

## Accessing the Login Page

1. Navigate to your organization's Biosafety Land URL 

> [!TIP]
> Your party 2 letter alpha country code can be found in the list of country codes at https://www.iso.org/obp/ui/#search.
> Or the assigned code given to a regional or special organization instance. For example:
> - For Canada, use [https://ca.bch.cbd.int/user/login](https://ca.bch.cbd.int/user/login)
> - For Switzerland, use [https://ch.bch.cbd.int/user/login](https://ch.bch.cbd.int/user/login)
> - For ASEAN, use [https://asean.bch.cbd.int/user/login](https://asean.bch.cbd.int/user/login)


2. Click the **Login** button in the top navigation bar

![Contributor Login](/images/biosafety-land/content/content-add-contributor-01-login.png)


## Login Process

1. Enter your **Email Address**
2. Enter your **Password**
3. Click the **Log in** button

::: tip
If you've forgotten your password, click the "Forgot password?" link to initiate a [password reset](/biosafety-land/authentication/password-reset).
:::

## First-Time Login

If this is your first time logging in:

1. You should have received an activation email with temporary credentials
2. Follow the [Account Activation](/biosafety-land/authentication/account-activation) process
3. Set up your permanent password

## Troubleshooting

### Invalid Credentials

If you receive an "Invalid credentials" error:

- Verify your username/email is correct
- Check that Caps Lock is not enabled
- Ensure you're using the correct password

### Account Locked

After multiple failed login attempts, your account may be temporarily locked for security purposes. Contact your system administrator if you need immediate access.

## Security Best Practices

- Never share your login credentials
- Use a strong, unique password
- Log out when using shared computers
- Report any suspicious activity to your administrator
