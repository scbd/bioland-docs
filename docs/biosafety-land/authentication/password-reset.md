# Password Reset

This guide explains how to reset your password if you've forgotten it or need to change it for security reasons.

## Forgot Your Password?

### Step 1: Request a Password Reset

1. Go to the login page
2. Look for the **"Forgot password?"** or **"Reset your password"** link below the login form
3. Click the link to open the password reset page

![Login Page with Forgot Password Link](/images/biosafety-land/authentication/password-reset-01-login-page.png)

4. Enter your registered email address in the form
5. Click **Submit** or **Email new password**

![Password Reset Request Form](/images/biosafety-land/authentication/password-reset-02-request-form.png)

![Email Address Entered](/images/biosafety-land/authentication/password-reset-03-email-entered.png)

### Step 2: Check Your Email

1. Check your inbox for a password reset email
2. The email will contain a secure reset link
3. Click the link to proceed

![Confirmation Message](/images/biosafety-land/authentication/password-reset-04-confirmation.png)

::: tip
Check your spam/junk folder if you don't see the email within a few minutes.
:::

### Step 3: Create a New Password

1. Enter your new password
2. Confirm the new password by entering it again
3. Click **Save** or **Reset Password**

## Password Requirements

Your new password must meet the following criteria:

- Minimum 8 characters
- At least one uppercase letter (A-Z)
- At least one lowercase letter (a-z)
- At least one number (0-9)
- At least one special character (!@#$%^&*)

## Changing Your Password (While Logged In)

If you're already logged in and want to change your password:

1. Click on your **user profile** icon
2. Select **Account Settings** or **Profile**
3. Navigate to the **Security** or **Password** section
4. Enter your current password
5. Enter and confirm your new password
6. Click **Save Changes**

## Reset Link Expired?

Password reset links expire after a set period for security. If your link has expired:

1. Return to the login page
2. Request a new password reset
3. Use the new link promptly

## Troubleshooting

### Not Receiving the Reset Email?

- Verify you're using the correct email address
- Check spam/junk folders
- Ensure your email inbox isn't full
- Contact your administrator for assistance

### Reset Link Not Working?

- Ensure you're clicking the most recent reset link
- Try copying and pasting the full URL
- Request a new reset link

## Security Tips

- Never share your password with anyone
- Use a unique password for this system
- Consider using a password manager
- Change your password regularly
