# Getting Started with Biosafety-Land

Welcome to Biosafety-Land! Follow these steps to get access and start using the application.

## Table of Contents

1. [Request Access](#request-access)
2. [Receive Login Credentials](#receive-login-credentials)
3. [Test Your Login](#test-your-login)
4. [Create Users](#create-users)

## Request Access

To get access to Biosafety-Land:

1. Contact your organization's secretariat
2. Request a Biosafety-Land account
3. Provide the following information:
   - Your full name
   - Email address
   - Department/Organization
   - Intended use case
4. Wait for approval (typically 1-3 business days)

::: tip Account Approval
Your request will be reviewed by the site administrator.
:::

## Receive Login Credentials

Once approved, you will receive:

- **Email notification** with your username
- **Temporary password** (you'll be prompted to change it on first login)
- **Application URL**: Your organization's Biosafety-Land instance (e.g., `https://biosafety.yourorganization.com`)
- **Support contact** information

## Test Your Login

1. Open your web browser
2. Navigate to the Biosafety-Land URL provided in your email
3. Enter your username and temporary password
4. Complete any additional authentication if required
5. Click **Sign In**
6. You'll be prompted to create a new password:
   - Choose a strong password
   - Confirm your new password
   - Click **Update Password**
7. You should now see the Biosafety-Land dashboard

### Troubleshooting Login Issues

- **Invalid credentials**: Double-check your username and password
- **Password expired**: Contact the secretariat for a password reset
- **Access denied**: Contact your site administrator
- **Cannot access URL**: Verify you're on the correct network/VPN if required
- **Other issues**: Contact your biosafety administrator

## Create Users

If you have administrative privileges, you can create accounts for other users:

1. Click on **Administration** in the main menu
2. Select **User Management**
3. Click **Create New User**
4. Fill in the user information:
   - Username
   - Email address
   - Full name
   - Role (Contributor, Content Manager, Site Manager)
5. Click **Create Account**
6. The new user will receive an email with their login credentials

### User Roles

- **Contributor**: Can create and edit their own content
- **Content Manager**: Can manage all content and media
- **Site Manager**: Can manage everything including users and configuration
