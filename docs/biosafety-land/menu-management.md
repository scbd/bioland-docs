# Menus

Menus help people move around your website. They show links to different pages. This guide shows you how to add, edit, and delete menus.

## Who Can Manage Menus?

| Role | Can Add Menus | Can Edit Menus | Can Delete Menus |
|------|---------------|----------------|------------------|
| **Site Manager** | ✅ Yes | ✅ Yes | ✅ Yes |
| **Content Manager** | ✅ Yes | ✅ Yes | ✅ Yes |
| **Contributor** | ❌ No | ❌ No | ❌ No |

**Note:** Only Site Managers and Content Managers can work with menus. Contributors cannot see or change menus.

## Adding a Menu

Menus organize the links on your website. You can add new menus for different parts of your site.

### What You Need to Know

- Menus have names
- Menus contain menu items (links)
- You can show menus in different places
- Menu items can link to pages or outside websites

### Content Manager Add Steps

#### Step 1: Open the Admin Area

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-01-admin-menu.png -->

1. Log in to your site
2. Click your name at the top right
3. Click "Content Management"

#### Step 2: Go to Structure

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-02-structure-menu.png -->

1. Look in the left menu
2. Find "Structure"
3. Click "Structure"

#### Step 3: Find Menus

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-03-menus-list.png -->

1. Look for "Menus" in the Structure section
2. Click "Menus"
3. You will see a list of all menus

#### Step 4: Start Adding a New Menu

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-04-add-button.png -->

1. Look at the top of the page
2. Find the "+ Add menu" button
3. Click the button

#### Step 5: Fill in Menu Details

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-05-menu-form.png -->

1. Type the menu name (like "Footer Menu")
2. Add a description (optional)
3. This helps you remember what the menu is for

#### Step 6: Configure Menu Settings

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-06-menu-settings.png -->

1. Choose where the menu appears
2. Set the menu language
3. Configure other options if needed

#### Step 7: Save the Menu

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-07-save-button.png -->

1. Check all your information
2. Find the "Save" button at the bottom
3. Click "Save"

#### Step 8: See Your New Menu

<!-- SCREENSHOT NEEDED: /images/biosafety-land/menus/menu-add-content-manager-08-success-message.png -->

1. You will see a success message
2. Your new menu appears in the list
3. Now you can add menu items to it

## Role Comparison

Different roles can do different things with menus.

| Action | Content Manager | Site Manager |
|--------|----------------|--------------|
| **Add Menu** | ✅ Yes (8 steps) | ✅ Yes (9 steps) |
| **Edit Menu** | ✅ Yes (8 steps) | ✅ Yes (9 steps) |
| **Delete Menu** | ✅ Yes (6 steps) | ✅ Yes (7 steps) |
| **Add Menu Items** | ✅ Yes | ✅ Yes |
| **Reorder Menu Items** | ✅ Yes | ✅ Yes |
| **Configure Permissions** | ❌ No | ✅ Yes |
| **Set Visibility Rules** | ❌ No | ✅ Yes |
| **See Advanced Settings** | ❌ No | ✅ Yes |

### Key Differences

1. **Content Manager:**
   - Can add, edit, and delete menus
   - Can add and arrange menu items
   - Cannot change permissions
   - Cannot set visibility rules
   - Focused on menu content

2. **Site Manager:**
   - Can do everything Content Managers can
   - Can set menu permissions
   - Can configure visibility rules
   - Sees advanced settings
   - Has full menu control

3. **Contributor:**
   - Cannot access menu management
   - Cannot see the menu section
   - Cannot make any menu changes

## Best Practices

### For Adding Menus

✅ **Do:**
- Use clear menu names
- Add descriptions to help others
- Plan your menu structure first
- Test menus on your site
- Create menus for specific purposes

❌ **Don't:**
- Use confusing names
- Create duplicate menus
- Skip the description field
- Make too many menus
- Use special characters in names

### For Editing Menus

✅ **Do:**
- Edit menus when structure changes
- Update descriptions when needed
- Test after making changes
- Keep menu names clear
- Check the live site after editing

❌ **Don't:**
- Change menus without checking first
- Edit menus during busy times
- Remove descriptions
- Make changes without testing
- Forget to save your changes

### For Deleting Menus

✅ **Do:**
- Check where the menu is used first
- Back up menu items if needed
- Verify the menu is not in use
- Delete test menus regularly
- Confirm you selected the right menu

❌ **Don't:**
- Delete menus that are in use
- Delete without checking first
- Rush the deletion process
- Delete system menus
- Delete without reading warnings

---

## Common Mistakes to Avoid

### When Adding Menus

1. **Creating duplicate menus**
   - Check existing menus first
   - Use search to find similar menus
   - Ask your team before creating new menus

2. **Using unclear names**
   - Names should explain the menu's purpose
   - Avoid abbreviations
   - Be specific

3. **Skipping descriptions**
   - Descriptions help others understand
   - Write clear, short descriptions
   - Explain what the menu is for

### When Editing Menus

1. **Not testing changes**
   - Always check the live site
   - Test all menu links
   - Verify menu appearance

2. **Editing wrong menu**
   - Check the menu name carefully
   - Verify you selected the correct one
   - Look at the description

3. **Forgetting to save**
   - Always click Save
   - Wait for the success message
   - Check that changes appear

### When Deleting Menus

1. **Deleting menus in use**
   - Check your website first
   - Look for menu usage
   - Ask before deleting

2. **Ignoring warnings**
   - Read all warning messages
   - Understand the impact
   - Cancel if unsure

3. **Deleting too quickly**
   - Take your time
   - Double-check your choice
   - Think before confirming

## Menu Items

After creating a menu, you need to add items to it. Menu items are the actual links.

### What Are Menu Items?

- Links in your menu
- Can point to pages on your site
- Can link to external websites
- Can have sub-items (nested menus)

### Adding Menu Items

1. Go to the menu list
2. Click "Edit menu" for your menu
3. Click "Add menu item"
4. Fill in the link details
5. Set the link title
6. Choose the parent item (for sub-menus)
7. Save the menu item

### Organizing Menu Items

- Drag and drop to reorder
- Indent items to create sub-menus
- Use clear link titles
- Keep menus simple and organized

## When to Create vs. Edit

### Create a New Menu When:
- You need a menu for a new section
- Your site structure changes
- You want to test a menu design
- Different parts need different menus

### Edit Existing Menu When:
- The menu name needs updating
- Settings need adjustment
- Menu location should change
- Permissions need updating

## Tips for Different Roles

### For Content Managers

- **Focus on menu content:** Add and organize menu items effectively
- **Keep it simple:** Don't overcomplicate menu structures
- **Test thoroughly:** Check menus on the live site
- **Use clear names:** Make menu names easy to understand
- **Ask for help:** Contact Site Manager for advanced settings

### For Site Managers

- **Plan carefully:** Think about site-wide menu strategy
- **Use permissions wisely:** Control who sees which menus
- **Document changes:** Note why you made menu changes
- **Review regularly:** Check menu structure periodically
- **Set visibility rules:** Control when and where menus appear
