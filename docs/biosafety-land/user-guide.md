# Biosafety-Land User Guide

This guide provides a quick overview of the most common tasks in Biosafety Land for **Content Managers**. Each section summarizes the key steps and links to detailed instructions for more information.

## Before You Begin

To use Biosafety Land, you need:

- A Content Manager account provided by your Site Manager or the CBD Secretariat
- Your login email and password
- A modern web browser (Chrome, Firefox, Safari, or Edge)

For a full list of roles and permissions, see [Access Levels](./authentication/access-levels.md).

---

## Logging In

To access the administration features, you must first log in.

1. Go to your Biosafety Land website (e.g., `https://your-country.bch.cbd.int`)
2. Click the **Login** link in the top navigation bar
3. Enter your **Username** and **Password**
4. Click the **Log in** button

![Login Page](/images/biosafety-land/user-guide/login-page.png)

After logging in, you will see a user icon in the top navigation bar. Clicking it opens a dropdown menu with links to **Publishing**, **Structure**, **Clear Cache**, **Edit Mode**, and **Logout**.

![User Menu Dropdown](/images/biosafety-land/user-guide/user-menu-dropdown.png)

::: tip
If you have forgotten your password, click the **Forgot password?** link on the login page.
:::

For detailed instructions, see [Login](./authentication/login.md).

---

## Logging Out

1. Click the **user icon** in the top-right corner of the navigation bar
2. Click **Logout** from the dropdown menu
3. Confirm by clicking **Log out** on the confirmation page

![Logout Confirmation](/images/biosafety-land/user-guide/logout-confirm.png)

::: warning
Any unsaved changes will be lost when you log out. Always save your work before logging out.
:::

For detailed instructions, see [Logout](./authentication/logout.md).

## Publishing Content

Content Managers can create, edit, delete, clone, and translate content. Biosafety Land supports many content types, including News, Announcements, Documents, Events, FAQs, and more.

### Adding New Content

1. Click the **user icon** in the top navigation bar
2. Click **Publishing** from the dropdown menu
3. You will see the content list with **Content**, **Media**, and **Unpublished** tabs — use the **Unpublished** tab to find drafts awaiting review or publication
4. Click the **+ Add content** button
5. Select a content type (e.g., News, Document, Event)
6. Fill in the required fields (Title, Body, etc.)
7. Set publishing options (Published/Unpublished, language, etc.)
8. Click **Save**

![Publishing Content List](/images/biosafety-land/user-guide/publishing-content-list.png)

### Editing Content

From the content list, find the item you want to edit and click the **Edit** button in the Operations column. Make your changes and click **Save**.

### Other Content Operations

| Operation | Description |
|-----------|-------------|
| **Clone** | Create a copy of existing content |
| **Delete** | Remove content from the site |
| **Translate** | Add translations in other languages |

For step-by-step instructions on all content operations, see [Content Management](./publishing/content/).

## Managing Media

Media includes images, videos, and documents that you upload to the site. Content Managers can add, edit, delete, and translate media.

### Supported File Types

| Type | Formats |
|------|---------|
| **Images** | JPG, PNG, GIF, SVG |
| **Videos** | MP4, WebM, OGV |
| **Documents** | PDF, DOC, DOCX, XLS, XLSX, TXT |

### Adding New Media

1. Click the **user icon** and select **Publishing**
2. Click the **Media** tab
3. Click the **+ Add media** button
4. Choose a media type (Image, Video, Document, etc.)
5. Upload your file
6. Fill in metadata (Name, Alt text, Description)
7. Set publishing options
8. Click **Save**

![Publishing Media List](/images/biosafety-land/user-guide/publishing-media-list.png)

::: tip
Use the **Table/Grid** toggle at the top of the media list to switch between a detailed table view and a visual thumbnail grid.
:::

For step-by-step instructions on all media operations, see [Media Management](./publishing/media/).

## Managing Menus

Menus organize the navigation links on your website. As a Content Manager, you can view and edit the links within existing menus.

### Accessing Menus

1. Click the **user icon** and select **Structure** from the dropdown
2. Click **Menus** on the Structure page
3. You will see the list of site menus (e.g., Footer, Resources, News & Updates)

![Structure Page](/images/biosafety-land/user-guide/structure-page.png)

![Menus List](/images/biosafety-land/user-guide/menus-list.png)

### Editing Menu Links

1. Click **List links** next to the menu you want to edit
2. Add, reorder, or remove menu items
3. Click **Save** when finished

::: info
The available menus are set up by Site Managers. Content Managers can edit the links within existing menus but **cannot** create or delete menus.
:::

For step-by-step instructions, see [Menu Management](./publishing/menus/).
