# Features

Discover the features available in Biosafety Land, the national-level platform for the Biosafety Clearing-House (BCH).

## Overview

Biosafety Land is a content management system designed for publishing and managing biosafety information related to Living Modified Organisms (LMOs) and the Cartagena Protocol on Biosafety. It provides tools for creating, organizing, and sharing biosafety-related content.

---

## Content Management

### Publishing Content

All three user roles can create and manage content in Biosafety Land.

- **Add Content**: Create new articles, pages, documents, and events
- **Edit Content**: Update existing content with new information
- **Delete Content**: Remove content that is no longer needed
- **Clone Content**: Duplicate existing content as a starting point
- **Translate Content**: Create multilingual versions of content

See [Content Publishing Guide](./publishing/content/) for detailed workflows.

### Content Types

Biosafety Land organizes information using structured content types:

| Content Type | Description |
|-------------|-------------|
| **Announcement** | Official notices, news releases, policy updates |
| **Capacity-Building** | Training materials, workshops, educational resources |
| **Contact** | National focal points, competent authorities, expert contacts |
| **Document** | Official reports, guidelines, manuals, legal documents |
| **FAQ** | Frequently asked questions and help content |
| **Field Trial** | LMO field trial information and monitoring reports |
| **Image or Video** | Photos, infographics, diagrams, video presentations |
| **Meeting or Event** | Conferences, workshops, webinars, training sessions |
| **National Information** | Country-specific biosafety information |
| **National Mainstreaming Strategy** | National biosafety integration strategies |
| **Project** | Research projects and implementation initiatives |
| **Related Website** | Links to external biosafety resources |
| **Status of LMO** | Regulatory decisions and approval status |

See [Content Types Reference](./context/content-types) for complete details.

## Media Management

### Managing Media Assets

Upload and organize images, videos, and documents.

- **Add Media**: Upload images, videos, and documents
- **Edit Media**: Update media metadata and descriptions
- **Delete Media**: Remove unused media files
- **Clone Media**: Duplicate media items
- **Translate Media**: Create translated versions of media

### Supported Formats

**Images:** JPG, JPEG, PNG, GIF, SVG  
**Videos:** MP4, WebM, OGV  
**Documents:** PDF, DOC, DOCX, XLS, XLSX, TXT

See [Media Management Guide](./publishing/media/) for detailed workflows.

## File Management

### Document Library

Content Managers and Site Managers can manage downloadable files.

- **Add Files**: Upload documents for user download
- **Edit Files**: Update file names and descriptions
- **Delete Files**: Remove files from the system

### Supported File Types

**Documents:** PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX  
**Archives:** ZIP, TAR, GZ  
**Text:** TXT, CSV

See [File Management Guide](./publishing/files/) for detailed workflows.

## Search

### Advanced Search

Biosafety Land features powerful search functionality.

- **Site Content Search**: Search local content with content type filtering
- **Secretariat Search**: Search the global CHM database via interoperability
- **Keyword Search**: Find content containing all specified terms
- **Phrase Search**: Use double-quotes for exact phrase matching
- **Real-time Results**: Results update as you type

### Search Tabs

| Tab | Description |
|-----|-------------|
| **Site Content** | Search local site with content type filters |
| **Secretariat** | Search global CHM database |

## User Roles & Permissions

### Three-Tier Role System

Biosafety Land uses a hierarchical permission system.

| Feature | Site Manager | Content Manager | Contributor |
|---------|:------------:|:---------------:|:-----------:|
| **Tools** | ✅ | ✅ | ✅ |
| **Publishing** | ✅ | ✅ | ✅ |
| **Structure** | ✅ | ✅ | ❌ |
| **Configuration** | ✅ | ❌ | ❌ |
| **People** | ✅ | ❌ | ❌ |
| **Reports** | ✅ | ❌ | ❌ |

See [Role Permissions](./administration/role-permissions) for complete details.

### Role Descriptions

**Contributor**
- Create and edit own content
- Add and manage media
- Basic publishing capabilities

**Content Manager**
- Manage all content and media
- Create and organize menus
- Manage site structure

**Site Manager**
- Full administrative access
- User management
- System configuration
- Site reports and analytics

## Administration

### People Management

Site Managers can manage all users in the system.

- **Add People**: Create new user accounts
- **Edit People**: Update user information and roles
- **Delete People**: Remove user accounts
- **Block/Unblock**: Control user access
- **Assign Roles**: Set user permission levels

See [People Management Guide](./administration/people) for detailed workflows.

### Configuration

Site Managers can configure system settings.

- **Site Information**: Name, slogan, email, front page
- **Regional Settings**: Date, time, language preferences
- **Media Settings**: File handling and upload configuration
- **User Settings**: Registration and account options
- **Security Settings**: Access and authentication controls
- **Email Settings**: Notification configuration

See [Configuration Guide](./administration/configuration) for detailed workflows.

### Reports

Site Managers can access site analytics and monitoring.

- **Content Reports**: Recent content, content by author
- **User Activity**: Who's online, top pages
- **Site Status**: System health and updates
- **Log Messages**: Recent activity and errors
- **Access Denied**: Security monitoring

See [Reports Guide](./administration/reports) for detailed workflows.
