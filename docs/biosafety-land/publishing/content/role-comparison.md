### Key Differences

**Contributor:**

- Can create and edit basic content
- Can only edit their own content
- Can only delete their own content
- Can only clone their own content
- Limited to standard fields
- Cannot change site structure
- No impact analysis before deleting
- Cannot clone published content
- Good for regular content creators

**Content Manager:**

- Can create and edit any content
- Can delete any content on the site
- Can clone any content from any user
- Can check content usage before deleting
- Can organize content in menus
- Can manage workflows
- Can change URL paths
- Can clean up after deletions
- Can change author on clones
- Can set clone language options
- Good for editorial staff

**Site Manager:**

- Full access to everything
- Can edit any content by any user
- Can delete with full impact analysis
- Can clone with all advanced options
- Can create backups before deleting
- Can configure advanced delete options
- Can set up redirects for deleted content
- Can change content type during clone
- Can set all permissions on clones
- Can configure URL patterns for clones
- Can change system settings
- Can manage all configurations
- Can revert to any old version
- Good for administrators
