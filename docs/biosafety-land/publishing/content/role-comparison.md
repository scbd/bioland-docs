## Role Comparison

This table shows what each role can do when adding, editing, deleting, cloning, and translating content.

### Content Creation and Editing Abilities

| Ability | Contributor | Content Manager | Site Manager |
|---------|:-----------:|:---------------:|:------------:|
| **Adding Content** | | | |
| Add content | ✅ | ✅ | ✅ |
| Add images to content | ✅ | ✅ | ✅ |
| Add files to content | ✅ | ✅ | ✅ |
| Save as draft | ✅ | ✅ | ✅ |
| Publish content | ❌ | ✅ | ✅ |
| Add to menus | ❌ | ✅ | ✅ |
| Set custom URLs | ✅ | ✅ | ✅ |
| Set redirect URLs | ❌ | ❌ | ✅ |
| **Editing Content** | | | |
| Edit own content | ✅ | ✅ | ✅ |
| Edit any content | ❌ | ✅ | ✅ |
| Add revision notes | ✅ | ✅ | ✅ |
| View revision history | ⚠️ Limited | ✅ | ✅ |
| Revert to old version | ❌ | ⚠️ Limited | ✅ |
| Update menu placement | ❌ | ✅ | ✅ |
| Change URL paths | ❌ | ✅ | ✅ |
| **Deleting Content** | | | |
| Delete own content | ✅ | ✅ | ✅ |
| Delete others' content | ❌ | ✅ | ✅ |
| Delete any content | ❌ | ✅ | ✅ |
| **Cloning Content** | | | |
| Clone own content | ✅ | ✅ | ✅ |
| Clone others' content | ✅ | ✅ | ✅ |
| Clone published content | ✅ | ✅ | ✅ |
| Clone draft content | ✅ | ✅ | ✅ |
| URL configuration | ✅ | ✅ | ✅ Advanced |
| **Translating Content** | | | |
| Translate own content | ✅ | ✅ | ✅ |
| Translate others' content | ❌ | ✅ | ✅ |
| Translate any content | ❌ | ✅ | ✅ |
| View translation status | ✅ | ✅ | ✅ |
| Edit existing translations | ⚠️ Own only | ✅ | ✅ |
| Delete translations | ⚠️ Own only | ✅ | ✅ |



**Legend:**

- ✅ = Yes, full access
- ⚠️ = Yes, but limited
- ❌ = No access

### Key Differences

**Contributor:**

- Can create and edit basic content
- Can only edit their own content
- Can only delete their own content
- **Cannot publish content directly** — content is saved as draft for review
- Can clone any published content (own or others')
- Limited to standard fields
- Cannot change site structure
- No impact analysis before deleting
- Good for regular content creators

**Content Manager:**

- Can create and edit any content
- Can delete any content on the site
- Can clone any content from any user
- Can check content usage before deleting
- Can organize content in menus
- Can manage workflows
- Can change URL paths
- Can clean up after deletions
- Can change author on clones
- Can set clone language options
- Good for editorial staff

**Site Manager:**

- Full access to everything
- Can edit any content by any user
- Can delete with full impact analysis
- Can clone with all advanced options
- Can create backups before deleting
- Can configure advanced delete options
- Can set up redirects for deleted content
- Can change content type during clone
- Can set all permissions on clones
- Can configure URL patterns for clones
- Can change system settings
- Can manage all configurations
- Can revert to any old version
- Good for administrators
