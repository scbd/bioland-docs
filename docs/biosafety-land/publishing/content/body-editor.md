# Body Editor Guide

The body editor is a rich text editor used throughout Biosafety Land for creating and editing content. This guide covers the main features and best practices.

## Editor Overview

The body editor (also called the WYSIWYG editor) allows you to:

- Format text with headings, bold, italic, and lists
- Insert links and images
- Create tables
- Embed media content
- Write structured content with proper formatting

## Basic Text Formatting

### Adding Headings

1. Place your cursor where you want the heading
2. Click the heading dropdown in the toolbar
3. Select **Heading 2**, **Heading 3**, or **Heading 4**
4. Type your heading text

::: tip
Use headings to structure your content. Start with Heading 2 for main sections, Heading 3 for subsections.
:::

### Creating Lists

**Bullet Lists:**
1. Click the bullet list icon in the toolbar
2. Type your first item
3. Press Enter to add more items
4. Press Enter twice to end the list

**Numbered Lists:**
1. Click the numbered list icon
2. Type your items
3. The numbers are added automatically

### Bold and Italic Text

- Select the text you want to format
- Click **B** for bold or *I* for italic
- Or use keyboard shortcuts:
  - Bold: `Ctrl+B` (Windows) or `Cmd+B` (Mac)
  - Italic: `Ctrl+I` (Windows) or `Cmd+I` (Mac)

## Inserting Links

1. Select the text you want to make a link
2. Click the link icon (ðŸ”—) in the toolbar
3. Enter the URL in the dialog
4. Click **Save** or **Insert**

### Link Types

- **External links**: Full URLs like `https://example.com`
- **Internal links**: Relative paths to other pages on the site
- **Email links**: Use `mailto:email@example.com`

## Inserting Images

1. Place your cursor where you want the image
2. Click the image icon in the toolbar
3. Choose an image from the media library or upload a new one
4. Add alt text (important for accessibility)
5. Click **Insert**

::: warning
Always add descriptive alt text to images for accessibility and SEO.
:::

## Writing Best Practices

### Structure Your Content

1. **Start with an introduction** - Explain what the content is about
2. **Use headings** - Break content into logical sections
3. **Keep paragraphs short** - 3-4 sentences per paragraph
4. **Use lists** - For steps, features, or options

### Write Clearly

- Use simple, direct language
- Avoid jargon unless necessary
- Define technical terms when first used
- Write for your audience

### Review Before Saving

Before saving your content:

1. Check spelling and grammar
2. Verify all links work
3. Ensure images display correctly
4. Preview on different screen sizes if possible

## Common Issues

### Text Not Formatting

If formatting isn't applying:
1. Make sure you've selected the text first
2. Try clicking the format button again
3. Clear formatting and reapply

### Images Not Displaying

If images don't appear:
1. Check the image was uploaded successfully
2. Verify the image path is correct
3. Ensure the image format is supported (JPG, PNG, GIF, WebP)

### Pasting from Word

When copying from Microsoft Word:
1. Use **Paste as Plain Text** if available
2. Or paste into a plain text editor first, then copy to the body editor
3. Reapply formatting after pasting
