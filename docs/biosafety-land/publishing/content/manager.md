## Content Manager Role

Content Managers have more options when adding content. They can manage any content on the site.

### Add Content - Content Manager {#add-content-content-manager}

Content Managers have more options when adding content. The basic steps are the same as Contributors.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Go to Content List

Navigate to the content management page:

```
/admin/content
```

::: tip Need help navigating?
See [Your Navigation Options](#your-navigation-options) for all the ways to reach this page.
:::

#### Step 2: Start Adding Content

1. Find the **Add Content** button
2. Click it to start the process

![Add Content](/images/biosafety-land/content/content-add-content-manager-02-add-button.png)

#### Step 3: Select Content Type

1. Pick the type of content you want to create
2. Content Managers can access all content types
3. Click on your chosen type

For details on each content type, see [Content Types](/biosafety-land/context/content-types).

![Select Type](/images/biosafety-land/content/content-add-content-manager-03-type-selection.png)

#### Step 4: Complete the Form

1. Fill in the title
2. Add your content in the body
3. Complete all required fields (marked with *)
4. Add optional fields as needed

![Complete Form](/images/biosafety-land/content/content-add-content-manager-04-news-form-top.png)

#### Step 5: Add Media and Files

1. Click **Add Media** if you want images
2. Click **Add File** for documents
3. Upload from your computer
4. Add descriptions for accessibility

![Add Media](/images/biosafety-land/content/content-add-content-manager-05-form-middle.png)

#### Step 6: Configure Advanced Settings

![Advanced Settings](/images/biosafety-land/content/content-add-content-manager-06-additional-fields.png)

Content Managers have extra options:

1. **Menu Settings** - Add content to menus
2. **URL Settings** - Create custom web addresses
3. **Revision Log** - Add notes about changes
4. **Language** - Set the content language
5. **Workflow** - Set review or approval steps

#### Step 7: Set Permissions

1. Choose who can view this content
2. Set editing permissions if needed
3. Configure workflow states

![Set Permissions](/images/biosafety-land/content/content-add-content-manager-07-publishing-options.png)

#### Step 8: Save and Publish

1. Review everything you entered
2. Click **Save and Publish** to make it live
3. Or click **Save** to keep it as a draft
4. You will see a success message

![Save and Publish](/images/biosafety-land/content/content-add-content-manager-08-save-button.png)

---

### Edit Content - Content Manager {#edit-content-content-manager}

Content Managers can edit any content on the site. They have more options than Contributors.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Go to Content List

Navigate to the content management page:

```
/admin/content
```

::: tip Need help navigating?
See [Your Navigation Options](#your-navigation-options) for all the ways to reach this page.
:::

#### Step 2: Search for Content

1. Use the search box to find content
2. Or use filters:
   - Filter by content type
   - Filter by status (published/draft)
   - Filter by author
   - Filter by date
3. Click **Apply** to filter

![Search Content](/images/biosafety-land/content/content-edit-content-manager-03-edit-form-top.png)

#### Step 3: Open Content for Editing

1. Find the content in the list
2. Click **Edit** button
3. The edit form will open

![Open for Editing](/images/biosafety-land/content/content-edit-content-manager-04-edit-form-middle.png)

#### Step 4: Make Your Changes

1. Update any field you need to change
2. Add or remove media
3. Update publishing settings
4. Change menu settings if needed

![Make Changes](/images/biosafety-land/content/content-edit-content-manager-05-additional-fields.png)

#### Step 5: Update Advanced Settings

![Advanced Settings](/images/biosafety-land/content/content-edit-content-manager-06-editing-changes.png)

Content Managers can change:

1. **Menu Links** - Update menu placement
2. **URL Path** - Change the web address
3. **Workflow State** - Move content through approval
4. **Language Settings** - Update translations
5. **Permissions** - Change who can view

#### Step 6: Add Detailed Revision Log

1. Scroll to **Revision Information**
2. Write a clear note about changes
3. Good notes help the team track updates

![Revision Log](/images/biosafety-land/content/content-edit-content-manager-07-revision-settings.png)

Examples:

- "Updated event date and location"
- "Added three new images"
- "Changed publishing status to draft"
- "Updated menu link placement"

#### Step 7: Choose Save Option

![Save Option](/images/biosafety-land/content/content-edit-content-manager-08-success-message.png)

Content Managers have multiple save options:

1. **Save** - Save changes as draft
2. **Save and Publish** - Make changes live now
3. **Save and Keep Unpublished** - Save but hide from public
4. Click your choice

#### Step 8: Verify Changes

1. After saving, check the success message
2. View the content to verify changes
3. Check that menus updated correctly
4. Test any links you changed

![Verify Changes](/images/biosafety-land/content/content-edit-content-manager-09-updated-content.png)

### Delete Content - Content Manager {#delete-content-content-manager}

Content Managers can delete any content on the site. They have more responsibility.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Go to Content List

Navigate to the content management page:

```
/admin/content
```

::: tip Need help navigating?
See [Your Navigation Options](#your-navigation-options) for all the ways to reach this page.
:::

#### Step 2: Search for Content to Delete

1. Use the search box to find content
2. Use filters to narrow results:
   - Filter by content type
   - Filter by status
   - Filter by author
   - Filter by date
3. Click **Apply** to filter

![Search Content](/images/biosafety-land/content/content-delete-content-manager-03-confirm-top.png)

#### Step 3: Check Content Usage

1. Before deleting, click on the content title
2. View where it's used
3. Check if it appears in menus
4. Check if other content links to it
5. Go back to the content list

![Check Usage](/images/biosafety-land/content/content-delete-content-manager-04-confirm-middle.png)

#### Step 4: Select Delete Option

1. Find the content in the list
2. Click the **Delete** button
3. The delete confirmation will appear

![Select Delete](/images/biosafety-land/content/content-delete-content-manager-05-translations-list.png)

#### Step 5: Review Delete Impact

1. Read the warning message
2. Check the impact summary
3. See what will be affected:

![Review Impact](/images/biosafety-land/content/content-delete-content-manager-06-success-message.png)

- Menu links that will break
- Pages that link to this content
- Related content that will be orphaned

#### Step 6: Add Deletion Note

1. Add a note explaining why you're deleting
2. This helps with record keeping
3. Good notes help your team understand

![Deletion Note](/images/biosafety-land/content/content-delete-content-manager-07-empty-list.png)

Example notes:

- "Content outdated and replaced by newer article"
- "Duplicate content - keeping only one version"
- "Requested by site manager for compliance"

#### Step 7: Confirm and Delete

1. Review all information
2. Click **Confirm Delete** to proceed
3. Or click **Cancel** to keep the content
4. Wait for the confirmation message

![Confirm Delete](/images/biosafety-land/content/content-delete-content-manager-08-all-users-content.png)

#### Step 8: Clean Up After Deletion

1. Check and remove broken menu links
2. Update pages that linked to deleted content
3. Redirect old URLs if needed
4. Notify team members if necessary

![Clean Up](/images/biosafety-land/content/content-delete-content-manager-09-delete-others-permission.png)

#### Best Practices for Content Managers

**Before Deleting:**

- Always check where content is used
- Save a backup copy if important
- Notify content authors
- Check with site manager if unsure
- Plan URL redirects

**After Deleting:**

- Update affected menus
- Fix broken links
- Set up 301 redirects
- Document the deletion
- Monitor for issues

### Clone Content - Content Manager {#clone-content-content-manager}

Content Managers have more clone options. They can clone any content on the site.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Go to Content List

Navigate to the content management page:

```
/admin/content
```

::: tip Need help navigating?
See [Your Navigation Options](#your-navigation-options) for all the ways to reach this page.
:::

#### Step 2: Find Content to Clone

1. Browse the content list
2. You can see all content (any author)
3. Use filters to find specific content
4. Use search to find content quickly

![Search & Select](/images/biosafety-land/content/content-clone-content-manager-03-clone-link.png)

#### Step 3: Access Clone Function

1. Click on the content you want to clone
2. Click on **Operations**
3. Select **Clone** from the menu

![Clone Options](/images/biosafety-land/content/content-clone-content-manager-04-clone-form-top.png)

#### Step 4: Configure Clone Options

1. Review all content fields
2. Decide what to keep or change
3. Set the status (Draft or Published)
4. Choose language options if needed

![Update Info](/images/biosafety-land/content/content-clone-content-manager-05-clone-form-middle.png)

#### Step 5: Set Clone Metadata

1. Update the title
2. Change author if needed
3. Set publish date
4. Update categories and tags
5. Adjust any taxonomy terms

![Advanced Clone Settings](/images/biosafety-land/content/content-clone-content-manager-06-editing-title.png)

#### Step 6: Edit Content Body

1. Review the body text
2. Make your changes
3. Update images if needed
4. Change links that should be different

![Workflow Settings](/images/biosafety-land/content/content-clone-content-manager-07-revision.png)

#### Step 7: Set Publishing Options

1. Choose if it should be published now
2. Set it as draft if not ready
3. Configure menu settings
4. Set URL path if needed

![Save Cloned](/images/biosafety-land/content/content-clone-content-manager-08-success-message.png)

#### Step 8: Save the Clone

1. Review all your changes
2. Click **Save** at the bottom
3. Wait for confirmation message
4. Your clone is now ready

![Confirm Clone](/images/biosafety-land/content/content-clone-content-manager-09-cloned-content-list.png)

#### What Content Managers Can Do

- Clone any content (any author)
- Clone published or draft content
- Change author on cloned content
- Set publishing options
- Clone with different language settings
- Bulk operations (limited)

### Translate Content - Content Manager {#translate-content-content-manager}

Content Managers can translate any content on the site. They have more translation options.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Go to Content List

Navigate to the content management page:

```
/admin/content
```

::: tip Need help navigating?
See [Your Navigation Options](#your-navigation-options) for all the ways to reach this page.
:::

#### Step 2: Find Content to Translate

1. Browse the content list
2. You can translate any content (any author)
3. Use filters to find specific content
4. Use search to find content quickly

![Search & Select](/images/biosafety-land/content/content-translate-content-manager-03-translations-page.png)

#### Step 3: Access Translate Function

1. Click on the content you want to translate
2. Click on **Translate** button
3. The translation dashboard opens

![Choose Language](/images/biosafety-land/content/content-translate-content-manager-04-add-translation.png)

#### Step 4: View Translation Status

1. See what languages already exist
2. See what languages are available
3. Check translation progress
4. See who created each translation

![Enter Translation](/images/biosafety-land/content/content-translate-content-manager-05-translation-form.png)

#### Step 5: Start New Translation

1. Find the target language
2. Click **Add** or **Edit** if translation exists
3. The translation editor opens
4. Original text shows for reference

![Advanced Options](/images/biosafety-land/content/content-translate-content-manager-06-translated-content.png)

#### Step 6: Translate All Fields

1. Translate the title
2. Translate body content
3. Translate meta descriptions
4. Translate image alt text
5. Translate menu links
6. Translate any custom fields

![Workflow Settings](/images/biosafety-land/content/content-translate-content-manager-07-save.png)

#### Step 7: Set Translation Settings

1. Choose if translation is complete
2. Set publishing status
3. Link to related translations
4. Set language-specific options

![Save Translation](/images/biosafety-land/content/content-translate-content-manager-08-success.png)

#### Step 8: Save or Publish Translation

1. Review all translated content
2. Choose **Save** to keep as draft
3. Or choose **Save and Publish** to make it live
4. Wait for confirmation message

![Verify Languages](/images/biosafety-land/content/content-translate-content-manager-09-publish-translation.png)

#### What Content Managers Can Do

- Translate any content from any author
- Edit existing translations
- Manage translation workflows
- Set language-specific options
- Link related translations
- Delete outdated translations
- Bulk translation operations (limited)
