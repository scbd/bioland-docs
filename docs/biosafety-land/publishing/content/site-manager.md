
## Site Manager Role

Site Managers have full control. They can add content and access all site features.

### Add Content - Site Manager {#add-content-site-manager}

Site Managers have full control. They can add content and access all site features.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Access Admin Toolbar

1. The admin toolbar is at the top
2. Site Managers see all six menu items
3. Click on **Publishing**

![Site Manager Toolbar](/images/biosafety-land/content/content-add-site-manager-02-toolbar.png)

#### Step 2: Navigate to Content

1. In the Publishing menu, click **Content**
2. You will see the content list page

![Navigate to Content](/images/biosafety-land/content/content-add-site-manager-03-content.png)

#### Step 3: Click Add Content

1. Find the **Add Content** button
2. Click to begin creating content

![Add Content Button](/images/biosafety-land/content/content-add-site-manager-04-add-button.png)

#### Step 4: Choose Any Content Type

Site Managers can create any content type:

- All standard content types
- Special administrative content
- Custom content types

1. Browse all available types
2. Click the one you need

For details on each content type, see [Content Types](/biosafety-land/context/content-types).

![Choose Content Type](/images/biosafety-land/content/content-add-site-manager-05-choose-type.png)

#### Step 5: Fill In Content Details

1. Enter the title
2. Add body content
3. Fill all required fields
4. Add any optional information

![Fill Details](/images/biosafety-land/content/content-add-site-manager-06-fill-details.png)

#### Step 6: Configure All Settings

![All Settings](/images/biosafety-land/content/content-add-site-manager-07-additional-fields.png)

Site Managers have access to all options:

1. **Basic Settings** - Title, body, summary
2. **Media** - Images and files
3. **Menu** - Add to any menu location
4. **URL** - Custom paths and redirects
5. **Workflow** - Full workflow control
6. **Publishing** - Date, status, permissions
7. **SEO** - Meta tags and descriptions
8. **Advanced** - Technical configurations

#### Step 7: Set Advanced Options

![Advanced Options](/images/biosafety-land/content/content-add-site-manager-08-publishing-options.png)

Site Managers can configure:

- Content type settings
- Field display options
- Revision tracking
- Translation settings
- Access restrictions

#### Step 8: Save Content

1. Review all settings
2. Choose save option:
   - **Save and Publish** - Make it live now
   - **Save** - Keep as draft
   - **Save and Configure** - Set more options
3. Click your choice
4. See the success message

![Save Content](/images/biosafety-land/content/content-add-site-manager-09-save-button.png)

### Delete Content - Site Manager {#delete-content-site-manager}

Site Managers have full deletion control. They must be very careful.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Access Content Management

1. Click **Publishing** in the admin toolbar
2. Select **Content**
3. The content list will open

![Access Content](/images/biosafety-land/content/content-delete-site-manager-02-operations-menu.png)

#### Step 2: Run Advanced Search

![Advanced Search](/images/biosafety-land/content/content-delete-site-manager-03-confirm-top.png)

Site Managers can use detailed filters:

1. **Text Search** - Find by title or content
2. **Content Type** - Filter by type
3. **Status** - Published, draft, or archived
4. **Author** - Filter by creator
5. **Date Range** - Find by creation or update date
6. **Language** - Filter by content language
7. **Tags/Categories** - Filter by taxonomy
8. Click **Filter** to apply

#### Step 3: Analyze Full Impact

![Analyze Impact](/images/biosafety-land/content/content-delete-site-manager-04-confirm-middle.png)

Before deleting, check all impacts:

1. Click the content title to view it
2. Check the **Content Usage** report:
   - All pages linking to this content
   - Menu items using this link
   - Related content references
   - Translation connections
   - Workflow dependencies
3. Note everything that will be affected

#### Step 4: Review System Dependencies

![System Dependencies](/images/biosafety-land/content/content-delete-site-manager-05-translations-list.png)

Check technical dependencies:

1. View blocks or widgets using this content
2. Check views or custom displays
3. Review API endpoints exposing this content
4. Check for automated processes
5. Review access permissions impact

#### Step 5: Create Backup

![Create Backup](/images/biosafety-land/content/content-delete-site-manager-06-success-message.png)

Before deleting important content:

1. Export content to backup
2. Download all associated media
3. Save revision history
4. Document all settings
5. Store backup securely

#### Step 6: Initiate Deletion Process

1. Return to content list
2. Select content to delete
3. Click **Delete** button
4. Advanced deletion dialog appears

![Initiate Delete](/images/biosafety-land/content/content-delete-site-manager-07-empty-list.png)

#### Step 7: Configure Deletion Options

![Delete Options](/images/biosafety-land/content/content-delete-site-manager-08-all-users-content.png)

Site Managers have advanced options:

1. **Delete Options:**
   - Delete content only
   - Delete content and revisions
   - Delete content and related translations
   - Delete content and associated media

2. **URL Handling:**
   - Create 301 redirect
   - Create 410 Gone status
   - No special handling

3. **Cleanup Options:**
   - Remove from all menus
   - Update related content
   - Clean up orphaned terms

4. Choose your options carefully

#### Step 8: Add Administrative Record

1. Add detailed deletion reason
2. Include approval information
3. Note who requested deletion
4. Document business justification
5. Add recovery instructions if needed

![Admin Record](/images/biosafety-land/content/content-delete-site-manager-09-delete-others-permission.png)

Example administrative notes:

- "Deleted per compliance audit - content contained outdated policy information"
- "Removed duplicate content created in error during migration"
- "Archived to comply with data retention policy - backed up to archive-2025-12"

#### Step 9: Execute and Verify Deletion

1. Review all settings one final time
2. Click **Confirm Deletion**
3. Wait for completion message
4. Verify deletion was successful:

![Verify Delete](/images/biosafety-land/content/content-delete-site-manager-10-full-control.png)

- Check content is removed
- Verify redirects work
- Test affected pages
- Confirm menu updates
- Check for broken links

#### Site Manager Best Practices

**Pre-Deletion Checklist:**

- ✅ Full impact analysis completed
- ✅ Backup created if content is important
- ✅ Team notified of upcoming deletion
- ✅ Redirects planned for URL
- ✅ Related content updated
- ✅ Menu links removed or updated
- ✅ Approval obtained if required

**Deletion Strategy:**

- Consider archiving instead of deleting
- Use workflow states (unpublish vs delete)
- Keep revisions for audit trail
- Document all major deletions
- Schedule deletions during low traffic
- Test in staging environment first

**Post-Deletion:**

- Monitor error logs for broken links
- Check analytics for 404 errors
- Update sitemaps
- Clear caches if needed
- Document completion
- Review any issues that arise

### Translate Content - Site Manager {#translate-content-site-manager}

Site Managers have full translation control. They can configure all language settings.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Access Admin Tools

1. Click **Publishing** in the toolbar
2. Click **Content** from the dropdown

![Access All Content](/images/biosafety-land/content/content-translate-site-manager-02-translate-tab.png)

#### Step 2: Find Content to Translate

1. View the complete content list
2. All content types are available
3. Use advanced filters
4. Search by any field
5. Sort by any column

![Advanced Search](/images/biosafety-land/content/content-translate-site-manager-03-translations-page.png)

#### Step 3: Open Translation Interface

1. Select the content to translate
2. Click **Translate**
3. Advanced translation dashboard opens
4. See full translation overview

![Select for Translation](/images/biosafety-land/content/content-translate-site-manager-04-add-translation.png)

#### Step 4: Configure Language Settings

![Translation Setup](/images/biosafety-land/content/content-translate-site-manager-05-translation-form.png)

Before translating, set options:

1. **Language Options:**
   - Source language
   - Target languages
   - Default language
   - Fallback behavior

2. **Translation Rules:**
   - Which fields to translate
   - Which fields to skip
   - Shared vs unique fields
   - Translation inheritance

#### Step 5: Start Translation

1. Choose target language
2. Click **Add Translation**
3. Advanced translation form opens
4. All configurable options available

![System Settings](/images/biosafety-land/content/content-translate-site-manager-06-translated-content.png)

#### Step 6: Translate Content

1. Translate all text fields
2. Update language-specific media
3. Adjust formatting for language
4. Translate metadata and SEO fields
5. Configure language-specific URLs

![Admin Options](/images/biosafety-land/content/content-translate-site-manager-07-save.png)

#### Step 7: Set Advanced Options

![Manage Languages](/images/biosafety-land/content/content-translate-site-manager-08-success.png)

Site Managers can configure:

1. **Translation Workflow:**
   - Review and approval states
   - Translation quality checks
   - Translator assignments

2. **Language-Specific Settings:**
   - Publishing schedules
   - Access permissions
   - Menu placements
   - URL structures

3. **Technical Options:**
   - Content negotiation
   - Language detection
   - Translation memory
   - Machine translation integration

#### Step 8: Link Translations

1. Link all language versions together
2. Set up language switcher
3. Configure cross-language relationships
4. Set canonical URLs

![Save All Translations](/images/biosafety-land/content/content-translate-site-manager-09-publish-translation.png)

#### Step 9: Save and Manage

1. Review all translation settings
2. Choose publishing option:
   - **Save** - Keep as draft
   - **Save and Publish** - Make live
   - **Save and Continue** - Translate more languages
3. Click to save
4. Verify translation works correctly

![Verify All Languages](/images/biosafety-land/content/content-translate-site-manager-10-language-settings.png)

#### What Site Managers Can Do

- Translate any content from any user
- Configure all language settings
- Set up translation workflows
- Manage translation memory
- Configure machine translation
- Set language-specific permissions
- Configure URL patterns per language
- Manage fallback languages
- Configure content negotiation
- Bulk translation operations
- Export/import translations
- Manage translation quality

#### Site Manager Translation Best Practices

**Before Translating:**

- Plan language strategy
- Configure language settings
- Set up translation workflows
- Train translators on system
- Test translation process

**During Translation:**

- Use translation memory
- Maintain consistency
- Follow style guides
- Review quality
- Test functionality

**After Translation:**

- Verify links work
- Test language switcher
- Check SEO settings
- Monitor analytics per language
- Gather feedback
