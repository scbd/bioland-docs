## Key Differences

### Adding Media

**Contributor:**
- Basic media upload
- Simple name and description
- No organizational features
- Fastest process (9 steps)

**Content Manager:**
- Full media upload
- Tags and collections
- Publishing control
- Medium process (10 steps)

**Site Manager:**
- Complete control
- All settings available
- Advanced configuration
- Full process (11 steps)

### Editing Media

**Contributor:**
- Can only edit own media
- Basic field updates
- No file replacement
- Quick process (7 steps)

**Content Manager:**
- Can edit all media
- Full field control
- Can replace files
- Publishing options (9 steps)

**Site Manager:**
- Complete editing access
- All field updates
- Advanced settings
- Full control (10 steps)

### Deleting Media

**Contributor:**
- Can only delete own media
- Simple confirmation
- No usage checking
- Quick process (6 steps)

**Content Manager:**
- Can delete all media
- Checks media usage first
- Can notify team
- Careful process (9 steps)

**Site Manager:**
- Complete deletion access
- Full usage review
- Permission checking
- Backup capability
- Most thorough (11 steps)

### Cloning Media

**Contributor:**
- Can clone any media
- Basic field updates
- Simple naming changes
- Quick process (8 steps)

**Content Manager:**
- Full cloning control
- Tags and collections
- Publishing options
- Medium process (9 steps)

**Site Manager:**
- Complete cloning access
- All settings available
- Advanced configuration
- Full process (10 steps)

### Translating Media

**Contributor:**
- Can translate own media
- Basic field translation
- Simple process
- Quick workflow (9 steps)

**Content Manager:**
- Can translate all media
- Full field translation
- Publishing options
- Language-specific files (10 steps)

**Site Manager:**
- Complete translation access
- All settings available
- Advanced configuration
- Full verification (12 steps)
