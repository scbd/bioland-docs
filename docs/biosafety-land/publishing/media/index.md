## Overview

All three roles can add media to Biosafety Land. The steps are similar for each role. This guide shows you how to manage media step by step.

### What You Need

- A user account (one of three roles)
- Access to the site at: https://e2e.bsl.staging.cbd.int/en
- Your login email and password
- Media files ready to upload (images, videos, or documents)

### Three User Roles

1. **Contributor** - Can add basic media
2. **Content Manager** - Can add and manage media
3. **Site Manager** - Can add media and manage the whole site
