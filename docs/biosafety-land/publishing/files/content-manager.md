## Content Manager Role

Content Managers can add, edit, and delete files. This section covers all file operations available to Content Managers.

### Add File - Content Manager {#add-file-content-manager}

Content Managers can add new files to the site. Here are the steps.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Open the Admin Menu

1. Click **Tools** in the top menu
2. The admin menu opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-02-admin-menu.png -->

#### Step 2: Go to Files

1. Click **Publishing** in the admin menu
2. Click **Files**
3. The Files page opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-03-files-list.png -->

#### Step 3: Start Adding a File

1. Click **Add file**
2. The Add File form opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-04-add-button.png -->

#### Step 4: Choose Your File

1. Click **Choose File** in the File upload field
2. Find your file on your computer
3. Select the file
4. Click **Open**
5. The file name appears next to the button

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-05-choose-file.png -->

#### Step 5: Enter File Name

1. Type a name in the **Name** field
2. Use a clear name that describes the file
3. Example: "Annual Report 2024"

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-06-name.png -->

#### Step 6: Add Description (Optional)

1. Scroll down to the **Description** field
2. Type a short description if needed
3. Example: "This report covers all activities in 2024"
4. Skip this if you don't need a description

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-07-description.png -->

#### Step 7: Save the File

1. Scroll to the bottom of the form
2. Click **Save**
3. Wait for the page to load
4. You see a success message

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-08-save.png -->

#### Step 8: Verify the File Was Added

1. Look for the success message at the top
2. You return to the Files list
3. Your new file appears in the list

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-add-content-manager-09-confirmation.png -->

### Edit File - Content Manager {#edit-file-content-manager}

Content Managers can edit existing files. Here are the steps.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Open the Admin Menu

1. Click **Tools** in the top menu
2. The admin menu opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-02-admin-menu.png -->

#### Step 2: Go to Files

1. Click **Publishing** in the admin menu
2. Click **Files**
3. The Files page opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-03-files-list.png -->

#### Step 3: Find the File to Edit

1. Look through the list of files
2. Find the file you want to edit
3. You can use filters to find it faster

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-04-find.png -->

#### Step 4: Open the Edit Form

1. Click **Edit** next to the file name
2. The Edit File form opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-05-edit-button.png -->

#### Step 5: Replace the File (Optional)

1. To replace the file, click **Remove**
2. Then click **Choose File**
3. Select a new file from your computer
4. Click **Open**
5. Skip this if you only want to change the name or description

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-06-replace-file.png -->

#### Step 6: Update the Name

1. Find the **Name** field
2. Change the name if needed
3. Keep it clear and descriptive

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-07-update-name.png -->

#### Step 7: Update Description (Optional)

1. Scroll to the **Description** field
2. Update the description if needed
3. Skip this if no changes are needed

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-08-description.png -->

#### Step 8: Save Your Changes

1. Scroll to the bottom of the form
2. Click **Save**
3. Wait for the page to load
4. You see a success message

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-09-save.png -->

#### Step 9: Verify Changes

1. Look for the success message at the top
2. You return to the Files list
3. Your changes appear in the list

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-edit-content-manager-10-confirmation.png -->

### Delete File - Content Manager {#delete-file-content-manager}

Content Managers can delete files. Here are the steps.

::: details Authenticated?
If you have not already logged in, follow the [Login](/biosafety-land/authentication/login) instructions.
:::

#### Step 1: Open the Admin Menu

1. Click **Tools** in the top menu
2. The admin menu opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-delete-content-manager-02-admin-menu.png -->

#### Step 2: Go to Files

1. Click **Publishing** in the admin menu
2. Click **Files**
3. The Files page opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-delete-content-manager-03-files-list.png -->

#### Step 3: Find the File to Delete

1. Look through the list of files
2. Find the file you want to delete
3. You can use filters to find it faster

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-delete-content-manager-04-find.png -->

#### Step 4: Open the Delete Confirmation

1. Click **Delete** next to the file name
2. A confirmation page opens

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-delete-content-manager-05-delete-button.png -->

#### Step 5: Review the Warning

1. Read the warning message carefully
2. Make sure this is the right file to delete
3. **Warning:** Deleting a file cannot be undone

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-delete-content-manager-06-warning.png -->

#### Step 6: Confirm Deletion

1. Click **Delete** to confirm
2. Wait for the page to load
3. You see a success message

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-delete-content-manager-07-confirm.png -->

#### Step 7: Verify Deletion

1. Look for the success message at the top
2. You return to the Files list
3. The deleted file no longer appears in the list

<!-- SCREENSHOT NEEDED: /images/biosafety-land/files/file-delete-content-manager-08-confirmation.png -->
