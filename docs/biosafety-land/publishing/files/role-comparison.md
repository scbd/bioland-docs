## Role Comparison

This table shows what each role can do with files.

| Action | Content Manager | Site Manager |
|--------|:---------------:|:------------:|
| **View files** | ✅ Yes | ✅ Yes |
| **Add files** | ✅ Yes | ✅ Yes |
| **Edit files** | ✅ Yes | ✅ Yes |
| **Replace file** | ✅ Yes | ✅ Yes |
| **Delete files** | ✅ Yes | ✅ Yes |
| **Configure advanced settings** | ❌ No | ✅ Yes |
| **View file usage** | ❌ No | ✅ Yes |
| **Manage all files** | ✅ Yes | ✅ Yes |

### Key Differences

**Content Manager:**
- Can manage all files
- Can add, edit, and delete files
- Cannot see advanced settings
- Cannot see where files are used

**Site Manager:**
- Can manage all files
- Has all Content Manager capabilities
- Can configure advanced settings
- Can see where files are used
- Can manage authoring information
- Can set URL aliases

**Contributors:**
- Cannot access file management at all
- Only Content Managers and Site Managers can work with files
