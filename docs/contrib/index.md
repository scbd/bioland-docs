# Contributing to Bioland Documentation

Welcome! This guide walks you through everything you need to know to contribute to the Bioland documentation. No programming experience or software installation is required — all editing is done directly in your web browser using GitHub.

## What You'll Learn

1. [Creating a GitHub Account](./01-github-account.md) — Set up your free GitHub account
2. [Finding and Navigating the Documentation](./02-navigating-repo.md) — Find the files you want to edit
3. [Understanding Branches](./03-branching.md) — How your changes stay safe until approved
4. [Editing Documentation Pages](./04-editing-pages.md) — Make changes using the web editor
5. [Previewing Your Changes](./05-previewing-changes.md) — Check your formatting before saving
6. [Working with Images](./06-images.md) — Upload and reference screenshots
7. [Creating a Pull Request](./07-pull-requests.md) — Submit your changes for review
8. [Markdown Quick Reference](./08-markdown-reference.md) — Formatting cheat sheet

## Before You Begin

You will need:
- A web browser (Chrome, Firefox, Safari, or Edge)
- An email address
- Access to the internet

That's it — no software to install, no commands to run.
